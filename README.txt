# MEMCREATOR_WEB


Memcreator_web - это социальная сеть для мемов, написанный на Python 3.10, при помощи flask 
____________________________________________________________________

Запуск:

Перед запуском распакуйте папку из .zip архива или "git clone https://github.com/KiraArensky/memcreators_web.git"
Установите все нужные библиотеки прописав команду "pip install -r requirements.txt" в консоли (терминале)
Запустите main.py 
Перейдите по полученной ссылке
____________________________________________________________________

Функционал:

Перед пользованием вы должны зарегистрироваться, после войти в аккаунт

У вас появился доступ к соцсети! Теперь вы можете выкладывать свои мемы в разделе "Создать пост". После выкладывания поста, он появится на вашей странице в разделе "Домашняя страница" и на стене в разделе "Лента"
____________________________________________________________________

Библиотеки:

Все используемы библиотеки есть в файле requirements.txt
