# MEMCREATOR_WEB


Memcreator_web - это сайт, написанный на Python 3.10, сделанный специально для приложения memcreator
____________________________________________________________________

Запуск:

Перед запуском распакуйте папку из .zip архива или "git clone https://github.com/KiraArensky/memcreators_web.git".
Установите все нужные библиотеки прописав команду "pip install -r requirements.txt" в консоли (терминале).
